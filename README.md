# MyCLI

A custom CLI tool built with Node.js and TypeScript.

## Features
- 10 CLI commands
- API integration (GitHub, Weather, Quotes)
- Colored output using chalk
- Help and version flags

## Installation

npm install
npm link

## Usage

mycli greet <name>
mycli calc <a> <b>
mycli github <username>
mycli weather <city>
mycli quote
